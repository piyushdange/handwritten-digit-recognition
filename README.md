# Handwritten Digit Recognition System

## 📌 Project Overview
This project uses a Convolutional Neural Network (CNN) to recognize handwritten digits (0–9).

## 📂 Dataset
- Self-created handwritten dataset
- 100 images (10 per digit)

## ⚙️ Technologies Used
- Python
- TensorFlow
- OpenCV
- Google Colab

## 🚀 Features
- Image preprocessing (grayscale, resize, normalization)
- CNN model training
- Live prediction using Gradio

## 📊 Results
- Achieved good accuracy on custom dataset
- Able to predict handwritten digits
